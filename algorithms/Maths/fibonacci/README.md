# Fibonacci Series
It is a series which proceeds by cumulative addition of consecutive terms
##0 1 1 2 3 5 8 13 ...

### Input Format
First line contains the number of terms you want to have in hte series.


### Output Format
The terms are displayed in every new line


### Sample Input
```
Enter no of times 5
```

### Sample Output
```
0
1
1
2
3
```
